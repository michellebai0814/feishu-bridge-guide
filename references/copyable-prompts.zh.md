# 可直接复制的话术

## 1. 让 agent 先分类再回答

```text
先别急着给我很多步骤。
先判断我这个问题更像下面哪一类：
1. 飞书聊天命令
2. 飞书后台权限
3. 飞书后台事件或回调
4. 本地配置文件
5. 本地资料或工作目录
6. 网页管理后台

然后只告诉我下一步先去哪里看。
```

## 2. 检查权限配置时

```text
你先不要默认是代码问题。
先帮我判断：我这个功能如果失效，更可能是权限、事件、回调，还是本地配置没开。
然后按优先级告诉我先检查哪 3 个点。
```

## 3. 做新手教学时

```text
请把这个功能讲给完全不懂的人听。
先说它属于哪一类，再说它要去哪里配置，最后给我一个最小可执行步骤。
```

## 4. 做 GitHub 教程时

```text
请把这段内容改成适合放进 GitHub README 的版本。
要求：
1. 普通人能看懂
2. 先讲它是什么，再讲去哪里配
3. 不要把不确定的内容说成唯一标准答案
```
