# 仓库边界说明

## 这个仓库是什么

- 这是一个给新手和 agent 使用的飞书 Bridge 教程型 Skill 包。
- 它的重点是“快速理解和正确分流”，不是完整底层源码讲解。
- 这套说明基于 `chenhg5/cc-connect` 搭建的 Bridge 使用场景整理而来。

## 这个仓库不是什么

- 它不是从零开发 Bridge 的源码仓库。
- 它不是飞书官方文档。
- 它也不是所有部署细节的完整百科。

## 推荐公开发布的核心内容

- `README.md`
- `QUICKSTART.zh.md`
- `SKILL.md`
- `agents/openai.yaml`
- `references/`
- `assets/images/`

## 不建议混进公开仓库的内容

- 本地调试脚本
- 监控脚本
- 个人环境缓存
- 不相关的测试产物
- 原始临时整理稿
